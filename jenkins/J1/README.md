# J1 – Jenkins Pipeline (Lab 6.3.6)

## Doel
Kennismaken met Jenkins pipelines en het definiëren van een
CI/CD workflow.

## Beschrijving
In dit experiment werd een declarative Jenkins pipeline opgesteld
met drie stages:
- Build
- Test
- Deploy

Elke stage bevat eenvoudige stappen om de werking van een pipeline
te demonstreren.

## Bestanden
- Jenkinsfile

## Uitvoering
De pipeline kan uitgevoerd worden in Jenkins door het Jenkinsfile
te koppelen aan een pipeline job.

## Resultaat / Opmerking
Door beperkingen van de labomgeving werd de pipeline niet effectief
uitgevoerd via de Jenkins UI.

De Jenkinsfile is correct opgesteld volgens de Jenkins declarative
pipeline syntax en toont inzicht in CI/CD concepten.
