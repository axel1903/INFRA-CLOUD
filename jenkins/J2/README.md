## Doel
Het zelfstandig uitbreiden van een Jenkins pipeline met extra functionaliteit.

## Beschrijving
In dit experiment werd een Jenkins declarative pipeline opgesteld met:
- pipeline parameters
- conditionele uitvoering van stages
- post actions

De pipeline simuleert een eenvoudige CI/CD workflow.

## Bestanden
- Jenkinsfile

## Uitvoering
De pipeline kan uitgevoerd worden in Jenkins door het Jenkinsfile
te koppelen aan een pipeline job.

## Resultaat / Opmerking
Door beperkingen van de labomgeving werd de pipeline niet effectief
uitgevoerd via de Jenkins UI.

De Jenkinsfile is correct opgesteld en toont inzicht in het uitbreiden
van Jenkins pipelines.
