## Doel
Het modelleren van een netwerkdevice en zijn interfaces
met behulp van YAML.

## Beschrijving
In dit experiment werd een YAML-bestand opgesteld dat
een netwerkdevice beschrijft met:
- algemene apparaatgegevens
- besturingssysteem
- meerdere netwerkinterfaces

Het YAML-model kan gebruikt worden als input voor
automatisatie of configuratiebeheer.

## Bestanden
- device.yml

## Resultaat
Het YAML-bestand beschrijft op een gestructureerde manier
een netwerkdevice en zijn interfaces.

Dit experiment toont inzicht in het gebruik van YAML
als dataformaat voor netwerkmodellen.
